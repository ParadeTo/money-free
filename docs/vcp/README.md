## VCP 模式简介

VCP 是由 Mark Minervini 提出的技术分析模式，用于识别股票价格波动性收缩的形态。该模式通常预示着股票可能突破并开始新的上升趋势。

### 核心特征

- **波动性收缩**：价格震荡幅度逐渐缩小
- **多次回调**：通常出现 2-4 次收缩回调
- **成交量萎缩**：在收缩过程中成交量逐渐减少
- **突破确认**：最终以放量突破确认形态完成
